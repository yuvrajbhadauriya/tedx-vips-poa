Custom Event Ticketing System

Project Overview

About the Project

This project is a complete, self-hosted ticketing system designed to be adaptable for any event. It provides a seamless and secure way to manage ticket sales without relying on third-party partners. The system consists of a public-facing customer portal for ticket applications and a secure, private admin dashboard for event organizers to verify payments, approve submissions, and manage the entire process from start to finish.

Why This System Was Built

Building a custom ticketing system from scratch offers several key advantages over using a pre-existing ticketing partner:

  • Full Control: Event organizers have complete ownership and control over the ticketing process, data, and branding.
  • Cost-Effective: It eliminates commission fees, directing all revenue to the event's bank account (saved ~â‚¹70,000 on the initial event).
  • Enhanced Security: Built with robust security features, including unique, unguessable QR codes for each ticket to prevent forgery.
  • Customized Workflow: The manual verification process is tailored specifically to the team's needs.
  • Professional Branding: Generated tickets can be custom-designed with the event's specific branding.

How It Works

For the Customer:

  1. Visits the public portal and sees the payment QR code.
  2. After paying, fills out a form with name, email, transaction ID, and a payment screenshot.
  3. Receives a "Thank You" and can track their ticket status.
  4. Once approved, receives a PDF ticket via email and can download it from the status page.

For the Admin Team:

  1. Logs into the secure Django Administration dashboard.
  2. Reviews new submissions with attendee details and payment screenshots.
  3. Approves a ticket by changing its status from "Pending" to "Approved".
  4. Selects approved submissions and uses a one-click action to email the secure PDF tickets.

Technology Stack

  • Frontend: HTML with Tailwind CSS
  • Database: SQLite (local) & PostgreSQL (production)
  • PDF Generation: reportlab
  • QR Code Generation: qrcode
  • Email Service: Gmail via Django SMTP
  • Hosting: PythonAnywhere
  • Version Control: Git & GitHub
